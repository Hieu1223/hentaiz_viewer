## About The Project

That evening I spent building a h*nt*i website in Flutter.

Features:
* Browse a grid of h*nt*i videos
* Pagination support
* Search videos without leaving the main page
* Watch videos inside the app
* Simple login/register system
* Comments for videos

### Built With

* Flutter
* Dart
* Provider
* HTTP
